regitlab 隐私权政策

最后更新日期：2026年1月

1. 概述

regitlab（以下简称"本插件"）是一款用于增强GitLab工作流程的浏览器扩展程序。本隐私权政策说明了我们如何收集、使用和保护您的信息。

2. 收集的信息

本插件收集以下信息，所有信息均存储在您的浏览器本地：

  • 邮箱地址：用于在GitLab中查询您的提交记录
  • 项目名称：用于执行GitLab相关操作（创建Pull Request、查询分支等）
  • Gemini API Key：用于调用Gemini API生成周报日志（由您自行提供）
  • 提示词模板：用于生成周报的提示词配置
  • 自定义URL按钮：您添加的快速跳转链接

重要说明：本插件还会临时读取页面中的用户信息（通过window.$udp.getUser()），但这些信息仅用于复制到剪贴板，不会存储或传输。

3. 数据存储

所有数据均使用Chrome浏览器的本地存储（chrome.storage.local）保存在您的设备上。数据不会上传到任何服务器，包括插件开发者的服务器。

4. 数据使用

收集的数据仅用于以下目的：

  • 自动创建GitLab Pull Request
  • 查询和显示您的GitLab提交记录
  • 生成周报日志（通过您提供的Gemini API Key调用Gemini服务）
  • 执行GitLab分支管理操作
  • 运行CI/CD流水线

5. 数据共享

本插件不会向任何第三方（包括插件开发者）共享、出售或传输您的数据。

当您使用Gemini API生成周报功能时，您的提交记录和提示词会发送到Gemini API服务（gemini-api.getan.edu.kg），这是您主动使用该功能时的行为，使用的是您自己提供的API Key。

6. 网络访问

本插件仅在以下域名下工作：

  • https://devops.cscec.com/*
  • https://dcs-uat-gray.cscec.com/*
  • https://gemini-api.getan.edu.kg/*（仅在使用Gemini功能时）

本插件不会访问其他网站，也不会收集或传输任何数据到插件开发者的服务器。

7. 数据安全

所有数据存储在您的浏览器本地，由Chrome浏览器提供安全保护。我们建议您：

  • 妥善保管您的Gemini API Key
  • 定期检查您的GitLab账户安全设置
  • 卸载插件时，本地存储的数据会被清除

8. 您的权利

您可以随时：

  • 在插件设置中查看、修改或删除您存储的数据
  • 卸载插件以删除所有本地存储的数据
  • 通过Chrome浏览器的扩展程序管理页面清除扩展程序数据

9. 第三方服务

本插件使用以下第三方服务：

  • Gemini API：当您使用周报生成功能时，数据会发送到Gemini API服务。请参考Google的隐私政策了解Gemini服务的数据处理方式。

10. 儿童隐私

本插件不面向13岁以下的儿童。我们不会故意收集儿童的个人信息。

11. 政策变更

我们可能会不时更新本隐私权政策。重大变更将在插件更新时通知您。继续使用本插件即表示您接受更新后的政策。
